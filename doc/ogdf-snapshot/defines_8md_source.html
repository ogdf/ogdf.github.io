3 # Compiler Defines {#defines}
4 
5 Here we list some of the available compiler definitions and macros.
6 
7 ## Defines set by OGDF
8 
9 | **Definition** | **Location** | **Description** |
10 |-------------------------------|--------------------|------------------|
11 | `OGDF_DEBUG` | build configuration | Perform OGDF assertions if compiled in Debug mode.
12 | `OGDF_HEAVY_DEBUG` | build configuration | Perform demanding OGDF assertions that might heavily increase runtime.
13 | `OGDF_USE_ASSERT_EXCEPTIONS` | build configuration | Throw exceptions instead of exiting by `assert()`.
14 | `OGDF_USE_ASSERT_EXCEPTIONS_WITH_STACK_TRACE` | build configuration | If assert exceptions are thrown, add a stack trace in the `what()` string.
15 | `OGDF_FUNCTION_NAME` | build configuration | Either `__func__` or `__PRETTY_FUNCTION__` if available. Only set if `OGDF_DEBUG` and `OGDF_USE_ASSERT_EXCEPTIONS` is set.
16 | `OGDF_DLL` | build configuration | Building or using ODFG as a DLL.
17 | `OGDF_INSTALL` | build configuration | Building ODFG as a DLL.
18 | `OGDF_MEMORY_POOL_TS` | build configuration | OGDF uses the custom thread-safe pool memory manager (default).
19 | `OGDF_MEMORY_POOL_NTS` | build configuration | OGDF uses the custom non-thread-safe pool memory manager.
20 | `OGDF_MEMORY_MALLOC_TS` | build configuration | OGDF uses the default c++ memory manager.
21 | `OGDF_HAS_LINUX_CPU_MACROS` | build configuration | Set if macros like `CPU_SET` are available.
22 | `OGDF_SSE3_EXTENSIONS` | build configuration | Set to the (system-specific) path of intrinsics (eg, `<intrin.h>`) or not defined.
23 | `COIN_OSI_CLP` | build configuration | Clp is the linear program solver (default).
24 | `COIN_OSI_GRB` | build configuration | Gurobi is the linear program solver.
25 | `COIN_OSI_CPX` | build configuration | CPLEX is the linear program solver.
26 | `OGDF_SYSTEM_WINDOWS` | `basic/internal/config.h` | Compiling for a Windows system.
27 | `OGDF_SYSTEM_UNIX` | `basic/internal/config.h` | Compiling for a UNIX system.
28 | `OGDF_SYSTEM_OSX` | `basic/internal/config.h` | Compiling for a Mac OS X system; in this case `OGDF_SYSTEM_UNIX` is also defined.
29 | `OGDF_ARCH_X86` | `basic/System.h` | Compiling for a 32-bit x86 (Intel/AMD) architecture.
30 | `OGDF_ARCH_X64` | `basic/System.h` | Compiling for a 64-bit x64 (Intel/AMD) architecture.
