Computes the angular resolution of the layout given by ga.

The angular resolution of a layout is the smallest angle formed by two edge segments that meet in a common endpoint or bend point. Angles are given in radians.

Parameters
gais the input layout.
pMaxAnglepoints to a variable in which the largest angle in the layout is stored; pMaxAngle may be a nullptr in which case nothing is stored.
pAvgAnglepoints to a variable in which the average angle in the layout is stored; pAvgAngle may be a nullptr in which case nothing is stored.
pStdDeviationpoints to a variable in which the standard deviation of angles is stored; pStdDeviation may be a nullptr in which case nothing is stored.
considerBendsdetermines whether bend points of edges shall be considered (true) or not (false).
Returns
the angular resolution (smallest angle) of the layout.

Computes the intersection graph H of the line segments in the layout given by ga.

The nodes of the intersection graph are all the endpoints of segments in the layout and intersection points. The edges corrsepond to edges in the input layout: If an edge connecting points p and q in H corresponds to an edge e in the input graph, then e contains a line segment s such that both p and q are endpoints or intersection points of s.

To put it more simple, we obtain graph H from the input layout ga by putting a dummy vertex on each crossing and bend point, and joining all nodes representing the same point in the plane.

Parameters
gais the input layout; if it contains bend points, edge segement of an edge's polyline is considered as line segment.
His assigned the intersection graph.
pointsmaps nodes in H to their geometric position in the layout.
origNodemaps nodes in H to nodes in the input graph (given by ga); points which are only intersection points of segments are mapped to nullptr.
origEdgemaps edges in H to the corresponding edges in the input graph (given by ga).

Computes the number of edge crossings in the layout given by ga.

If several edge segments cross in the same point, this is counted as if all of these segments would cross pairwise. E.g., if three edge segments cross in a common points, this counts as three crossings.

Parameters
gais the input layout; if it contains bend points, edge segement of an edge's polyline is considered as line segment.
