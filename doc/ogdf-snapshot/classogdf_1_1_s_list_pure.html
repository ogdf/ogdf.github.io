Detailed Description

template<class E>
class ogdf::SListPure< E >

Singly linked lists.

Elements of the list are instances of type SListElement. Use SListConstIterator or SListIterator in order to iterate over the list.

In contrast to SList, instances of SListPure do not store the length of the list.

Template Parameters
Eis the data type stored in list elements.
