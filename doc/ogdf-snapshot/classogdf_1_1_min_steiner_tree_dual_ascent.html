Detailed Description

template<typename T>
class ogdf::MinSteinerTreeDualAscent< T >

Dual ascent approximation algorithm for Steiner tree problems.

The algorithm is implemented following the paper by Richard T. DualAscent, "A Dual Ascent Approach for Steiner Tree Problems on a Directed Graph", Mathematical Programming 28, pages 271-287, 1984.
