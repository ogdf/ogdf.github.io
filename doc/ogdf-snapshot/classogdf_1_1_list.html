Detailed Description

template<class E>
class ogdf::List< E >

Doubly linked lists (maintaining the length of the list).

Elements of the list are instances of type ListElement. Use ListConstIterator or ListIterator in order to iterate over the list.

In contrast to ListPure, instances of List store the length of the list.

Template Parameters
Eis the data type stored in list elements.
