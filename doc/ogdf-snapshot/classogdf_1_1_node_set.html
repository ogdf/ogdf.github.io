Detailed Description

Node sets.

A node set maintains a subset S of the nodes contained in an associated graph. This kind of node set provides efficient operations for testing membership, insertion and deletion of elements, and clearing the set.

In contrast to NodeSetPure, a NodeSet provides efficient access to the number of nodes stored in the set.

See also
  • NodeSetPure, NodeSetSimple
  • FaceSet, FaceSetPure, FaceSetSimple

§ clear()

void ogdf::NodeSet::clear ( )
inline

Removes all nodes from S.

After this operation, S is empty and still associated with the same graph. The runtime of this operations is O(k), where k is the number of nodes in S before this operation.
