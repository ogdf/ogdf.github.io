§ validate()

int ogdf::OgmlParser::validate ( const XmlTagObject *  xmlTag,
int  ogmlTag 
)
private

Checks if all tags (XmlTagObject), their attributes (XmlAttributeObject) and their values are valid (are tags expected, do they own the rigth attributes...) and sets a valid flag to these.

Furthermore it checks if ids of tags are unique and if id references are valid. See OgmlTag.h for semantics of the encodings. Returns the validity state of the current processed tag.
