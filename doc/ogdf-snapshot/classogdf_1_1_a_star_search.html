Detailed Description

template<typename T, typename H = a_star_internal::UninformedHeuristic<T>>
class ogdf::AStarSearch< T, H >

A-Star informed search algorithm.

The algorithm is a generalization the the shortest path algorithm by Dijkstra. It was first described in "A Formal Basis for the Heuristic Determination of Minimum Cost Paths" by Hart, Nilsson and Raphael in 1968.

The algorithms yields an optimal solution to the single pair shortest pair problem. A heuristic for calculating a lower bound on the distance from any node to the target is required. The algorithm can also be used to compute approximate solutions at a faster pace.

Template Parameters
TThe type of edge cost
HThe heuristic to be used. Defaults to a heuristic which always returns null, thus behaving like ogdf::Dijkstra.
