Detailed Description

template<class E>
class ogdf::SList< E >

Singly linked lists (maintaining the length of the list).

Elements of the list are instances of type SListElement. Use SListConstIterator<E> or SListIterator in order to iterate over the list. In contrast to SListPure, instances of SList store the length of the list and thus allow constant time access to the length.

Template Parameters
Eis the data type stored in list elements.
