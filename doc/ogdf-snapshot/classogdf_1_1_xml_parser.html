Detailed Description

This class parses the XML input file and builds up a parse tree with linked elements XMLTagObject and XMLAttributeObject.

The class XmlScanner is used to get the token for the parse process.

) const

Traverses the parseTree starting at startTag using the path description in path, which contains the infoIndices of the tags which have to be traversed.

If the XmlTagObject associated to the last infoIndex in the path is found, it is returned via targetTag and the return value is true If the XmlTagObject is not found the return value is false.
