Detailed Description

template<class T>
class ogdf::EmbedderMaxFaceBiconnectedGraphsLayers< T >

Embedder that maximizes the external face (plus layers approach).

Precondition
Input graphs need to be biconnected.

See the paper "Graph Embedding with Minimum Depth and Maximum External Face" by C. Gutwenger and P. Mutzel (2004) for details. The algorithm for maximum external face is combined with the algorithm for maximum external layers which defines how to embed blocks into inner faces. See diploma thesis "Algorithmen zur Bestimmung von guten Graph-Einbettungen fr orthogonale Zeichnungen" (in German) by Thorsten Kerkhof (2007) for details.
